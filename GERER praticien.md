\- En tant que visiteur médical délégué ou responsable de secteur, je veux consulter/créer/modifier des praticiens de la région ou du secteur.



Le délégué/responsables se connecte



1- le délégué demande à gérer les praticiens.

&nbsp;  -> clique sur "praticiens"

&nbsp;  

&nbsp;  2- le système invite à sélectionner le nom du praticien dans une liste déroulante comprenant

      tous les praticiens de la région puis valide.

&nbsp;     \*\*affiche tous les praticiens (il peut trier par ordre alphabétique ou par ordre de numéro, il peut aussi afficher par region/secteur en cliquant sur "Modifables")\*\* 

&nbsp;     -> il sélectionne un praticien (exemple: Notini Alain (n°1))

          2-a Le délégué demande à créer un nouveau praticien. le système affiche un écran vierge

&nbsp;             ->clique sur "+ Créer un nouveau praticien" 

&nbsp;             \*\*affiche un formulaire VIDE avec "\*" (pour marque les champs obligatoires)\*\*

          2-a-1 Le délégué saisi le praticien sans choisir le type de praticien                

                2-a.2 le système l'en informe et lui demande de confirmer

&nbsp;                     \*\*quand un champs obligatoire est vide un texte apparait\*\*

                      2-a.3 le visiteur confirme : aller en 3

&nbsp;                           ->clique sur le bouton "valider" en vert

                      2-a.3-a le visiteur ne confirme pas : retour en 2

&nbsp;                             ->clique sur le bouton "annuler" en rouge 

&nbsp;                             \*\*retour sur la page "Gérer praticien"\*\*



3 le système affiche un écran de saisie pré rempli avec le numéro et nom du praticien et

&nbsp; toutes les informations concernant le praticien, le type de praticien et les spécialités du

&nbsp; praticien.

&nbsp; ->clique sur "Modifiables" sélectionne un praticien de sa région/secteur

&nbsp; \*\*affiche seulement les praticiens de sa région/secteur qu'il peut modifier\*\*



4- Le délégué saisit ses informations.

    4-b Le délégué enregistre le praticien sans saisir les spécialités

        4-b.1 le système l'en informe et lui demande de confirmer

            4-b.2 le délégué confirme : aller en 4

            4-b.2-a le délégué ne confirme pas : retour en 3



5 – le délégué valide les informations en cliquant sur le bouton valider

    5-a le visiteur a valider et une information obligatoire n'a pas été saisie : le système affiche

        un message d'erreur qui indique l'information manquante. Retour en 4.



6- le système enregistre les données.
