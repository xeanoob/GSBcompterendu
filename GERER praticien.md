\- En tant que visiteur médical délégué ou responsable de secteur, je veux consulter/créer/modifier des praticiens de la région ou du secteur.



Le délégué/ responsables se connecte



1- le délégué demande à gérer les praticiens.

-> clique sur "praticiens"



2- le système invite à sélectionner le nom du praticien dans une liste déroulante comprenant

&nbsp;  tous les praticiens de la région puis valide.



&nbsp;   2-a Le délégué demande à créer un nouveau praticien. le système affiche un écran vierge

&nbsp;       2-a-1 Le délégué saisi le praticien sans choisir le type de praticien

&nbsp;           2-a.2 le système l'en informe et lui demande de confirmer

&nbsp;               2-a.3 le visiteur confirme : aller en 3

&nbsp;               2-a.3-a le visiteur ne confirme pas : retour en 2



3 le système affiche un écran de saisie pré rempli avec le numéro et nom du praticien et

toutes les informations concernant le praticien, le type de praticien et les spécialités du

praticien



4- Le délégué saisit ses informations.

&nbsp;   4-b Le délégué enregistre le praticien sans saisir les spécialités

&nbsp;       4-b.1 le système l'en informe et lui demande de confirmer

&nbsp;           4-b.2 le délégué confirme : aller en 4

&nbsp;           4-b.2-a le délégué ne confirme pas : retour en 3



5 – le délégué valide les informations en cliquant sur le bouton valider

&nbsp;   5-a le visiteur a valider et une information obligatoire n'a pas été saisie : le système affiche

&nbsp;       un message d'erreur qui indique l'information manquante. Retour en 4.



6- le système enregistre les données.
